Remote PC Controller

Control your friend's PC remotely.

View on GitHub Download .zip Download .tar.gz

Remote PC Controller is done as an education project trying to replicate the TeamViewer software. From here on we will mention Remote PC Controller as RPC.

You can use RPC to control one PC from another one where both of them are connected via LAN.
